# Dice Game

## Description
A simple dice game application built with web technologies.

## Features
- Roll dice with randomized outcomes
- Track scores and game history
- User-friendly interface
- Responsive design

## Installation
1. Clone or download the repository
2. Navigate to the project directory
3. Open `index.html` in your web browser

## Usage
1. Click the "Roll Dice" button to generate a random number
2. View your score and results on screen
3. Play multiple rounds as desired

## Technologies Used
- HTML5
- CSS3
- JavaScript
